## 功能介绍 

 
<img width="430" height="430" alt="image" src="https://github.com/user-attachments/assets/9e28c49f-07ce-4c64-9049-a1187845127e" />


本小程序是为博物馆打造的轻量化服务平台，旨在提升观众参观体验、优化博物馆运营效率，实现线上线下服务闭环。它既为观众提供便捷的信息获取、参观与活动预约、个人中心管理等功能，也为博物馆管理员提供高效的内容维护、预约管理、用户管理及数据导出等后台能力。对观众：信息获取更及时，参观体验更流畅，个人服务管理更便捷。
对博物馆：运营管理更高效，数据统计更精准，服务质量可量化。

## 技术运用
- 前端基于微信小程序平台进行开发
- 后端基于Java Springboot架构开发
- 数据库： MySQL (8.0+)
